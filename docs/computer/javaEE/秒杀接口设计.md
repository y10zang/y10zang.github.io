# 秒杀系统设计

link: `https://mp.weixin.qq.com/s/mDN6fM69UDLq4TGoZmGryA?`

### 前言

秒杀系统是高并发场景的典型代表，涉及缓存、消息队列、限流、分布式锁等多种技术。本文将从零开始设计一个完整的秒杀系统。

### 一、秒杀系统的特点

#### 1.1 业务特点

高并发：

• 瞬时流量巨大（10万+ QPS）

• 读多写少（99%的请求是查询）

• 库存有限（商品数量远小于用户数）

时间集中：

• 流量在特定时间点爆发

• 需要精确的时间控制

用户体验：

• 响应时间要快（< 100ms）

• 不能超卖

• 公平性（先到先得）

#### 1.2 技术挑战

挑战1：高并发读

- 大量用户查询商品信息
- 数据库压力巨大

挑战2：高并发写

- 大量用户同时下单
- 库存扣减并发问题

挑战3：超卖问题

- 库存100件，卖出101件
- 需要保证数据一致性

挑战4：恶意请求

- 机器人刷单
- 需要防刷机制

### 二、系统架构设计

#### 2.1 整体架构

```
用户请求
    ↓
CDN（静态资源）
    ↓
Nginx（负载均衡 + 限流）
    ↓
网关（鉴权 + 限流 + 熔断）
    ↓
秒杀服务（业务逻辑）
    ↓
Redis（缓存 + 库存）
    ↓
消息队列（异步处理）
    ↓
订单服务（创建订单）
    ↓
MySQL（持久化）
```

#### 2.2 核心流程

```java
// 秒杀流程
1. 用户点击秒杀按钮
2. 前端防重复提交
3. 后端接口限流
4. 验证用户资格（登录、黑名单）
5. 验证商品状态（是否开始、是否结束）
6. 扣减Redis库存
7. 发送消息到MQ
8. 异步创建订单
9. 返回结果给用户
```

### 四、核心功能实现

#### 4.1 商品详情查询（缓存优化）

```java
@Service
@Slf4j
public class SeckillProductService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Autowired
    private SeckillProductMapper productMapper;
    
    private static final String PRODUCT_CACHE_KEY = "seckill:product:";
    private static final long CACHE_EXPIRE = 3600; // 1小时
    
    /**
     * 获取秒杀商品详情
     */
    public SeckillProduct getProduct(Long productId) {
        String key = PRODUCT_CACHE_KEY + productId;
        
        // 1. 从缓存获取
        SeckillProduct product = (SeckillProduct) redisTemplate.opsForValue().get(key);
        if (product != null) {
            log.info("缓存命中：{}", productId);
            return product;
        }
        
        // 2. 缓存未命中，查询数据库
        product = productMapper.selectById(productId);
        if (product == null) {
            // 缓存空对象，防止缓存穿透
            redisTemplate.opsForValue().set(key, new SeckillProduct(), 60, TimeUnit.SECONDS);
            return null;
        }
        
        // 3. 写入缓存
        redisTemplate.opsForValue().set(key, product, CACHE_EXPIRE, TimeUnit.SECONDS);
        
        return product;
    }
    
    /**
     * 预热：将商品信息加载到缓存
     */
    @Scheduled(cron = "0 0 * * * ?") // 每小时执行一次
    public void warmUpCache() {
        log.info("开始预热缓存");
        
        // 查询即将开始的秒杀商品
        List<SeckillProduct> products = productMapper.selectUpcoming();
        
        for (SeckillProduct product : products) {
            String key = PRODUCT_CACHE_KEY + product.getId();
            redisTemplate.opsForValue().set(key, product, CACHE_EXPIRE, TimeUnit.SECONDS);
            
            // 预热库存到Redis
            String stockKey = "seckill:stock:" + product.getId();
            redisTemplate.opsForValue().set(stockKey, product.getStock());
        }
        
        log.info("缓存预热完成，商品数量：{}", products.size());
    }
}
```

#### 4.2 库存扣减（Redis + Lua脚本）

```java
@Service
@Slf4j
public class StockService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    private static final String STOCK_KEY = "seckill:stock:";
    
    /**
     * Lua脚本：原子性扣减库存
     */
    private static final String DEDUCT_STOCK_SCRIPT = 
        "local stock = redis.call('get', KEYS[1])\n" +
        "if not stock then\n" +
        "    return -1\n" +
        "end\n" +
        "if tonumber(stock) <= 0 then\n" +
        "    return 0\n" +
        "end\n" +
        "redis.call('decr', KEYS[1])\n" +
        "return 1";
    
    /**
     * 扣减库存
     * @return 1-成功 0-库存不足 -1-商品不存在
     */
    public int deductStock(Long productId) {
        String key = STOCK_KEY + productId;
        
        // 执行Lua脚本
        DefaultRedisScript<Long> script = new DefaultRedisScript<>();
        script.setScriptText(DEDUCT_STOCK_SCRIPT);
        script.setResultType(Long.class);
        
        Long result = redisTemplate.execute(script, 
            Collections.singletonList(key));
        
        return result != null ? result.intValue() : -1;
    }
    
    /**
     * 初始化库存
     */
    public void initStock(Long productId, Integer stock) {
        String key = STOCK_KEY + productId;
        redisTemplate.opsForValue().set(key, stock);
    }
    
    /**
     * 获取当前库存
     */
    public Integer getStock(Long productId) {
        String key = STOCK_KEY + productId;
        Object stock = redisTemplate.opsForValue().get(key);
        return stock != null ? Integer.parseInt(stock.toString()) : 0;
    }
}
```

#### 4.3 防止重复购买（分布式锁）

```java
@Service
@Slf4j
public class SeckillService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Autowired
    private StockService stockService;
    
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    private static final String USER_LOCK_KEY = "seckill:lock:";
    private static final String USER_ORDER_KEY = "seckill:order:";
    
    /**
     * 秒杀下单
     */
    public SeckillResult seckill(Long userId, Long productId) {
        // 1. 检查用户是否已经购买过
        String orderKey = USER_ORDER_KEY + userId + ":" + productId;
        Boolean hasOrdered = redisTemplate.hasKey(orderKey);
        if (Boolean.TRUE.equals(hasOrdered)) {
            return SeckillResult.fail("您已经购买过该商品");
        }
        
        // 2. 获取分布式锁，防止重复下单
        String lockKey = USER_LOCK_KEY + userId + ":" + productId;
        Boolean locked = redisTemplate.opsForValue()
            .setIfAbsent(lockKey, "1", 10, TimeUnit.SECONDS);
        
        if (Boolean.FALSE.equals(locked)) {
            return SeckillResult.fail("请勿重复提交");
        }
        
        try {
            // 3. 扣减库存
            int result = stockService.deductStock(productId);
            
            if (result == -1) {
                return SeckillResult.fail("商品不存在");
            }
            if (result == 0) {
                return SeckillResult.fail("商品已售罄");
            }
            
            // 4. 标记用户已购买
            redisTemplate.opsForValue().set(orderKey, "1", 24, TimeUnit.HOURS);
            
            // 5. 发送消息到MQ，异步创建订单
            SeckillMessage message = new SeckillMessage();
            message.setUserId(userId);
            message.setProductId(productId);
            message.setTimestamp(System.currentTimeMillis());
            
            rabbitTemplate.convertAndSend("seckill.exchange", 
                "seckill.order", message);
            
            log.info("秒杀成功，用户：{}，商品：{}", userId, productId);
            return SeckillResult.success("秒杀成功，正在生成订单");
            
        } finally {
            // 6. 释放锁
            redisTemplate.delete(lockKey);
        }
    }
}
```

```java
@Service
@Slf4j
public class SeckillOrderService {
    
    @Autowired
    private SeckillOrderMapper orderMapper;
    
    @Autowired
    private SeckillProductService productService;
    
    @Transactional(rollbackFor = Exception.class)
    public SeckillOrder createOrder(Long userId, Long productId) {
        // 1. 查询商品信息
        SeckillProduct product = productService.getProduct(productId);
        if (product == null) {
            throw new BusinessException("商品不存在");
        }
        
        // 2. 检查是否已经有订单
        SeckillOrder existOrder = orderMapper.selectByUserAndProduct(userId, productId);
        if (existOrder != null) {
            log.warn("订单已存在：{}", existOrder.getOrderNo());
            return existOrder;
        }
        
        // 3. 创建订单
        SeckillOrder order = new SeckillOrder();
        order.setOrderNo(generateOrderNo());
        order.setUserId(userId);
        order.setProductId(productId);
        order.setProductName(product.getProductName());
        order.setPrice(product.getPrice());
        order.setQuantity(1);
        order.setStatus(0); // 待支付
        
        orderMapper.insert(order);
        
        log.info("订单创建成功：{}", order.getOrderNo());
        return order;
    }
    
    private String generateOrderNo() {
        // 订单号生成规则：时间戳 + 随机数
        return System.currentTimeMillis() + "" + 
            (int)((Math.random() * 9 + 1) * 100000);
    }
}
```

#### 5.1 接口限流（令牌桶算法）

```java
@Aspect
@Component
@Slf4j
public class RateLimitAspect {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Around("@annotation(rateLimit)")
    public Object rateLimit(ProceedingJoinPoint joinPoint, RateLimit rateLimit) 
            throws Throwable {
        
        String key = "rate:limit:" + rateLimit.key();
        int limit = rateLimit.limit();
        int period = rateLimit.period();
        
        // 使用Redis实现令牌桶
        Long count = redisTemplate.opsForValue().increment(key);
        
        if (count == 1) {
            // 第一次访问，设置过期时间
            redisTemplate.expire(key, period, TimeUnit.SECONDS);
        }
        
        if (count > limit) {
            log.warn("触发限流：{}, 当前请求数：{}", key, count);
            throw new RateLimitException("请求过于频繁，请稍后再试");
        }
        
        return joinPoint.proceed();
    }
}
```

#### 5.2 用户限流（滑动窗口）

```java
@Service
@Slf4j
public class UserRateLimitService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    /**
     * 检查用户是否超过限流
     * @param userId 用户ID
     * @param limit 限制次数
     * @param window 时间窗口（秒）
     * @return true-允许访问 false-限流
     */
    public boolean checkUserLimit(Long userId, int limit, int window) {
        String key = "user:limit:" + userId;
        long now = System.currentTimeMillis();
        long windowStart = now - window * 1000;
        
        // 移除窗口外的记录
        redisTemplate.opsForZSet().removeRangeByScore(key, 0, windowStart);
        
        // 获取当前窗口内的请求数
        Long count = redisTemplate.opsForZSet().zCard(key);
        
        if (count != null && count >= limit) {
            log.warn("用户 {} 触发限流，当前请求数：{}", userId, count);
            return false;
        }
        
        // 添加当前请求
        redisTemplate.opsForZSet().add(key, String.valueOf(now), now);
        
        // 设置过期时间
        redisTemplate.expire(key, window, TimeUnit.SECONDS);
        
        return true;
    }
}
```

#### 5.3 验证码防刷

```java
@Service
@Slf4j
public class CaptchaService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    private static final String CAPTCHA_KEY = "captcha:";
    
    /**
     * 生成验证码
     */
    public String generateCaptcha(String sessionId) {
        // 生成4位数字验证码
        String code = String.format("%04d", new Random().nextInt(10000));
        
        // 存入Redis，5分钟过期
        String key = CAPTCHA_KEY + sessionId;
        redisTemplate.opsForValue().set(key, code, 5, TimeUnit.MINUTES);
        
        return code;
    }
    
    /**
     * 验证验证码
     */
    public boolean verifyCaptcha(String sessionId, String code) {
        String key = CAPTCHA_KEY + sessionId;
        String cachedCode = (String) redisTemplate.opsForValue().get(key);
        
        if (cachedCode == null) {
            return false;
        }
        
        // 验证后删除
        redisTemplate.delete(key);
        
        return cachedCode.equals(code);
    }
}
```

#### 6.2 倒计时与自动刷新

```js
export default {
  data() {
    return {
      product: null,
      countdown: 0,
      status: 'NOT_START' // NOT_START, IN_PROGRESS, ENDED
    }
  },
  mounted() {
    this.loadProduct()
    this.startCountdown()
  },
  methods: {
    async loadProduct() {
      const result = await api.getProduct(this.productId)
      this.product = result.data
      this.updateStatus()
    },
    
    updateStatus() {
      const now = Date.now()
      const startTime = new Date(this.product.startTime).getTime()
      const endTime = new Date(this.product.endTime).getTime()
      
      if (now < startTime) {
        this.status = 'NOT_START'
        this.countdown = Math.floor((startTime - now) / 1000)
      } else if (now >= startTime && now < endTime) {
        this.status = 'IN_PROGRESS'
        this.countdown = Math.floor((endTime - now) / 1000)
      } else {
        this.status = 'ENDED'
        this.countdown = 0
      }
    },
    
    startCountdown() {
      setInterval(() => {
        if (this.countdown > 0) {
          this.countdown--
          
          // 倒计时结束，刷新状态
          if (this.countdown === 0) {
            this.loadProduct()
          }
        }
      }, 1000)
    }
  }
}
```

```yml
      # 响应时间过长告警
```
